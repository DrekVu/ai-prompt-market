# AI Prompt Marketplace

Decentralized marketplace for AI prompts & datasets on Shelby (Aptos blockchain).

## Quick Start

1. `pnpm install`
2. `pnpm dev`

Open http://localhost:3000. Connect Petra wallet to test upload/search.

## Features
- Upload prompts/datasets to Shelby.
- On-chain metadata with price.
- Search & buy with micropayment.
